Developer tool

Website-agnostic scraper inspector

This is separate from the CareerLens user app. When a scraper runs with visual mode enabled, the actual target website opens in Chrome and the scraper highlights the elements it reads, clicks, waits for, and scrolls to.

Back to CareerLens

Run visible scraping

Actual website, actual scraper logic
CLI resume-build scrape --visual --delay-ms 900

The shared Playwright debug layer is website-agnostic. Scraper code calls the same highlight helper with a selector and label, so Facebook, LinkedIn, or any future source can show the developer what is being inspected without adding product-facing UI.

Environment controls

Developer only
Variable Purpose Default Scope
RESUME_BUILD_PLAYWRIGHT_VISUAL Turns on the highlighted browser run. 0 debug
RESUME_BUILD_PLAYWRIGHT_DELAY_MS Slows each Playwright step so the logic is visible. 700 debug
RESUME_BUILD_PLAYWRIGHT_HIGHLIGHT_MS Controls how long selector outlines remain visible. 900 debug
